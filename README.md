# BackupKit Static Starter

This is a plain HTML/CSS/JS starter structure for a landing page and basic site pages.

## Structure

- `index.html`: Homepage
- `about.html`: About page
- `contact.html`: Contact page
- `assets/css/main.css`: Shared styles
- `assets/js/main.js`: Shared behavior
- `assets/img/`: Image assets
- `assets/icons/`: Icon assets
- `sections/`: Reusable HTML section templates

## Run locally

Open `index.html` directly in your browser, or serve this folder with any static server.
